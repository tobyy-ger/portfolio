O projektu


Tento projekt vznikl jako závěrečná práce do školního předmětu zaměřeného na tvorbu webových aplikací. Hlavním cílem bylo vytvořit funkční a přehledný konvertor měn pomocí čistého HTML, CSS a JavaScriptu.


Jak funguje JavaScript konvertor

JavaScript soubor zajišťuje veškerou logiku převodu měn. Níže je stručný popis toho, jak vše funguje:

  1. Kurzy měn: Na začátku je definovaný objekt rates, který obsahuje pevně nastavené kurzy mezi jednotlivými měnami. Například: 
    EUR: { USD: 1.1, CZK: 24.5, GBP: 0.88 }
    To znamená, že 1 EUR = 1.1 USD, nebo 1 EUR = 24.5 CZK.
  2. Zachycení kliknutí na tlačítko: Pomocí .addEventListener čeká na kliknutí na tlačítko „Převést“: 
    document.getElementById('convertBtn').addEventListener('click', () => { ... });
  3. Načtení vstupních dat: Po kliknutí skript vezme hodnoty ze vstupních polí:
    • amount – částka, kterou chce uživatel převést
    • from – měna, ze které se převádí
    • to – cílová měna
  4. Ověření vstupu: Skript kontroluje, jestli je částka číslo a větší než nula. Pokud ne, vypíše chybu "Zadej platnou částku větší než 0.".
  5. Samotný převod: Pokud jsou vstupy v pořádku a měny nejsou stejné, provede se výpočet: 
    const convertedAmount = amount * rates[from][to];
    Výsledek se pak zobrazí ve formátu
    "Výsledek: 123.45 USD"
  6. Výjimky: Pokud není dostupný kurz mezi dvěma měnami, skript ohlásí, že kurz není dostupný.


Použité technologie

  • HTML5 – základní struktura stránky
  • CSS3 – stylování, včetně responzivního designu a efektů
  • JavaScriptVýpočty, zajištění chybových stavů atd.
